## 📋 ROADMAP HỌC TẬP

### Cấp độ 1: Beginner (Chưa biết gì)

1. Đọc **[START_HERE.md](START_HERE.md)** → Chạy thử
2. Đọc **[TUTORIAL.md](TUTORIAL.md)** → Hiểu code
3. Chạy notebooks trong `notebooks/01_eda_exploration.ipynb`

**Thời gian:** 1-2 giờ

---

### Cấp độ 2: Intermediate (Biết Python cơ bản)

1. Đọc **[README.md](README.md)** → Hiểu tổng quan
2. Đọc **[PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md)** → Hiểu cấu trúc
3. Chạy **[START_HERE.md](START_HERE.md)** → Setup
4. Khám phá code trong `src/` folders
5. Chạy tất cả notebooks
6. Thử modify hyperparameters trong `configs/config.yaml`

**Thời gian:** 3-4 giờ

---

### Cấp độ 3: Advanced (Biết ML)

1. Clone repo → Setup ngay
2. Đọc code trong `src/`
3. Chạy `python scripts/train.py`
4. Đọc **[docs/CONFIGURATION.md](docs/CONFIGURATION.md)**
5. Thử các models khác (XGBoost, LightGBM)
6. Hyperparameter tuning
7. Deploy model

**Thời gian:** 2-3 giờ
